# Smart-bin

Using ESP32 microcontrollers, ultrasonic sensors, and tri-color LEDs, we transformed ordinary bins into intelligent nodes that monitor fill levels in real-time, display status at a glance (🟢<50%, 🟡50-80%, 🔴>80%), and offer touchless lid opening for improved hygiene. The system transmits data via MQTT over WiFi to the school's existing cloud infrastructure, where a Python subscriber feeds a PostgreSQL database and a Streamlit dashboard provides facilities managers with live updates, smart routing maps, and automatic alerts when bins reach 80% capacity. This hybrid edge-cloud architecture enables real-time decisions (<2s latency) at the bin level while enabling long-term analytics in the cloud. Beyond the technical achievement, the prototype delivered measurable environmental benefits: 33% reduction in plastic bag usage through need-based replacements, 60% fewer collection trips (reducing CO₂ emissions), and 100% prevention of overflow incidents during testing. The project demonstrates that low-cost, sustainable IoT solutions can be built using existing campus infrastructure, open-source technologies, and student-driven innovation proving that smart campuses don't require enterprise budgets.
